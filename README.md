# Shoot Help Program

### 프로그램 제작 이유
이 프로그램은 어쩌구 저쩌구 해서 만들게 되었습니다.

### 프로그램 로직 설명
이 프로그램은 어떻게 저렇게 요렇게 동작합니다.

### 핵심 알고리즘
이 프로그램에서 손과 팔을 영상 처리하여 어쩌구 저쩌구

### 프로그램의 장점
이 프로그램을 쓰면 이런 분야, 저런 사람, 요런 경우에 좋음

### 사용 방법
이 프로그램을 사용하려면 이렇게 시작해서 요렇게 진행하고 이러면 마무리 됩니다.
